## サイト概要
議論をすることをメインとしたSNS

### サイトテーマ
きになったことは、はなしあおう

### テーマを選んだ理由
TwitterやFacebookのような既存のSNSは、特定の投稿の一部が注目されたりして議論の過程が見づらかったりする。
そう感じたため、議論を成立させるためには、違った形のSNSの需要があるかもしれないと感じたため。

### ターゲットユーザ
20代30代


### 主な利用シーン
政治について議論
わからないことについて議論
ディスカッションの授業の一環として学生が使う

など
